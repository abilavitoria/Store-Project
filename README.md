Passos para iniciar o projeto
1. no pgAdmin 4 crie um banco de dados com o seguinte nome store_project_bd,
2. clone o repositório do github no github desktop,
3. no IntelliJ idea selecione a opção Open -> Github -> Store-Project -> back-end,
4. após isso selecione src -> main -> java -> Application, no simbulo play ▶ ele inicai a aplicação,
5. agora vamos no VsCode, selecionamos a opção Open folder -> Github -> Store-Project -> front-end -> UserInterface -> Select folder,
6. minha recomendação é que você começe pela página de cadastro (cadastroPage.html),
7. você precisa realizar um cadastro inicial para fazer o login, após o primeiro cadastro não é necessário efetuá-lo novamente,
8. se você se cadastrar com um cpf você terá permissões de vendedor, se você quiser receber permissões de admin tem que se cadastrar com cnpj,
9. após o login você será direcionado a página inicial e poderá efetuar (dependendo de suas permissões) cadastro, edição e exclusão de clientes,
produtos e vendas.
